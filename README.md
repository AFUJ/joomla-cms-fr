## 2- Qu'est-ce que Joomla?
* Joomla! est un système de gestion de contenu (CMS) qui vous permet de créer des sites web et des applications en ligne puissantes.
* C'est un logiciel libre et Open Source, distribué sous la licence publique générale GNU version 2 ou ultérieure.
* Il s'agit d'une application serveur web simple et puissante qui nécessite un serveur avec PHP et soit MySQL, PostgreSQL ou SQL Server pour fonctionner.
Vous pouvez trouver les exigences techniques complètes ici : [https://downloads.joomla.org/technical-requirements](https://downloads.joomla.org/technical-requirements).
